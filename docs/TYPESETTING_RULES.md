# 排版规则说明文档

本文档定义智能 Word 排版 Agent 的排版规则，涵盖各级标题、正文、强调、小字题注等元素。

---

| 范围标识 | 说明 | 对应 Word 样式 |
|----------|------|----------------|
| `heading_1` | 一级标题 | Heading 1 / 标题 1 |
| `heading_2` | 二级标题 | Heading 2 / 标题 2 |
| `heading_3` | 三级标题 | Heading 3 / 标题 3 |
| `heading_4` | 四级标题 | Heading 4 / 标题 4 |
| `heading_5` | 五级标题 | Heading 5 / 标题 5 |
| `heading_6` | 六级标题 | Heading 6 / 标题 6 |
| `heading_7` | 七级标题 | Heading 7 / 标题 7 |
| `heading_8` | 八级标题 | Heading 8 / 标题 8 |
| `heading_9` | 九级标题 | Heading 9 / 标题 9 |
| `body` | 正文 | Normal / 正文 |
| `emphasis` | 强调（加粗/斜体） | 正文内 run 级样式 |

## 二、各级标题规范

### 2.1 公文标准（GB/T 9704-2012）

| 级别 | 字体 | 字号 | 对齐 | 段前/段后 |
|------|------|------|------|-----------|
| 一级标题 | 黑体 | 22pt（二号） | 居中 | 段前 12pt，段后 12pt |
| 二级标题 | 黑体 | 16pt（三号） | 左对齐 | 段前 6pt，段后 6pt |
| 三级标题 | 仿宋 | 16pt（三号） | 左对齐 | 段前 6pt，段后 6pt |
| 四级及以下 | 仿宋 | 14pt（四号） | 左对齐 | 段前 3pt，段后 3pt |

## 三、正文规范

### 3.1 公文正文

- **字体**：仿宋（或仿宋_GB2312）
- **字号**：12pt（小四号）或 16pt（三号），视单位要求
- **首行缩进**：2 字符（约 24pt）
- **行距**：1.5 倍
- **对齐**：两端对齐（justify）

### 3.2 论文正文

- **中文字体**：宋体
- **西文字体**：Times New Roman
- **字号**：12pt（小四号）
- **首行缩进**：2 字符（约 24pt）
- **行距**：1.5 倍
- **段前/段后**：各 6pt
- **对齐**：两端对齐

### 3.3 默认正文

- **字体**：宋体
- **字号**：12pt
- **首行缩进**：2 字符
- **行距**：单倍
- **对齐**：左对齐

## 四、强调规范

强调用于正文中的重点内容，通常通过加粗、斜体或颜色实现。

| 类型 | 说明 | 典型用法 |
|------|------|----------|
| **加粗** | `bold: true` | 关键词、术语、重要概念 |
| **斜体** | `italic: true` | 外文术语、书名、强调语气 |
| **颜色** | `color: "#FF0000"` | 批注、修订、特殊标记 |

**注意**：强调为 run 级样式，作用于正文内的部分文字，而非整段。

---

## 五、小字题注规范

小字题注用于图表说明、脚注、注释等，字号小于正文。

### 5.1 图表题注

- **字体**：宋体或仿宋
- **字号**：10.5pt（五号）或 9pt（小五）
- **对齐**：居中（图题）或左对齐（表题）
- **段前/段后**：3pt

### 5.2 脚注

- **字体**：宋体
- **字号**：9pt（小五）
- **首行缩进**：0 或 1 字符

### 5.3 引用说明

- **字体**：仿宋或楷体
- **字号**：10.5pt（五号）
- **缩进**：左右各缩进 2 字符

## 六、预设与 JSON 配置

详细规则可参见项目根目录下的 `typesetting_rules.json`，该文件可被程序读取并应用。

使用示例：

```python
from src.main import process_document

# 使用公文预设
process_document("report.docx", "", preset="official")

# 使用论文预设
process_document("thesis.docx", "正文小四宋体", preset="thesis")
```

## 七、首行缩进换算

- 1 字符 ≈ 12pt（以五号字宽为基准）
- 2 字符 ≈ 24pt
- 3 字符 ≈ 36pt
